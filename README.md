# Phonebook backend

Backend for a phonebook application, using Express and MongoDB

### Packages
- *nodemon* for node server auto-refresh
- *Express* as framework
  - *Express* JSON parser
- *Morgan* with `tiny` config as middleware
- *CORS* middleware
- *Static* middleware for frontend integration
